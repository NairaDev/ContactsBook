1.Добавление контакта: Функция добавления контакта 
  реализована с использованием AJAX-запроса.
  При отправке формы добавления нового контакта данные формы
  передаются на сервер с помощью AJAX, что позволяет добавить
  контакт без перезагрузки страницы. При успешном добавлении
  контакта отображается сообщение "Контакт успешно добавлен!" 
  без перезагрузки страницы.

2.Удаление контакта: Функция удаления контакта также
  реализована с использованием AJAX-запроса. При нажатии на 
  опцию удаления контакта в списке, AJAX-запрос отправляется на
  сервер, чтобы удалить контакт из базы данных. После успешного 
  удаления контакта отображается сообщение "Контакт успешно 
  удален!" без перезагрузки страницы.

3.Если контакт с таким телефонным номером уже существует в 
  базе данных, выводится сообщение
  "Контакт с таким телефонным номером уже существует!".

4.Если поля формы заполнены некорректно, то выводится сообщение
  "Пожалуйста, заполните все поля в правильном формате!".
   Валидация реолизована с помощю regexp, праильный формат для
   телефонного номера взята из указонного вами макета.
